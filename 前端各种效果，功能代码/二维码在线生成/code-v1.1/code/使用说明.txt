使用说明：

1、解压缩后，把文件放到网站任意目录即可。

2、支持中文、英文二维码生成。

3、随着信息量的增大，二维码图片的尺寸也会变大，出于系统资源的考虑，增加了输入的字节数限制，默认为360字节。

4、修改t.txt文件、png文件夹为777权限，否则会提示生成失败。
