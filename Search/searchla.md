<mark>搜索技巧</mark>  
1. 把口语化搜索变成关键字搜索  
       note: 能减少冗余信息  
2. 限定在某个网站中  
       example: 美国大选 site:xinhuanet.com  
3. 特定时间内的关键词搜索  
       example: 只是付费 2015..2018  
4. 不包含某些关键词  
      example: 注册会计师 -推广 -推广链接  
5. 将某些关键词包含在标题中  
      example: 刘亦菲 intitle:Dior  
6. 筛选文件类型  
      note: filetype: pdf
